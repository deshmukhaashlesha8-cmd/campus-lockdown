# 🎮 CampusLockdown

## 📖 About the Project

CampusQuest is a 2D top-down exploration game developed as a team project. Players explore a college campus, interact with NPCs, complete missions, and experience campus life through an engaging adventure.

The game is being designed with future support for both **Windows** and **Android**.

---

## 🎯 Project Objective

The objective of CampusQuest is to create an interactive virtual college where players can:

- 🏫 Explore the campus and different floors
- 💻 Visit computer labs and classrooms
- 📚 Access the library
- 👨‍🏫 Interact with teachers and students
- 📋 Complete missions and tasks
- 💾 Save their game progress
- 🎮 Enjoy a fun and educational experience

---

## 🚀 Planned Features

- Top-down 2D gameplay
- Smooth player movement
- Interactive NPCs
- Mission and quest system
- Dialogue system
- Multiple college floors
- Library and laboratories
- Main menu and settings
- Save & Load system
- Background music and sound effects
- Android support (planned)
